# 217. Contains Duplicate

https://leetcode.com/problems/contains-duplicate/


# Tests

```
Input: nums = [1, 2, 3, 1]
Output: True
```

```
Input: nums = [1, 2]
Output: False
``` 


# Solutions

| Solution            | Time complexity  | Space complexity              |
|---------------------|------------------|-------------------------------|
| **Brute Force**     | `O(n^2)`         | `O(1)`                        |
| **Sort**            | `O(nlogn)`       | `O(n)` or `O(1)` if optimized |
| **HashSet**         | `O(n)`           | `O(n)`                        |
| **Lengths**         | `O(1)`           | `O(1)`                        |
