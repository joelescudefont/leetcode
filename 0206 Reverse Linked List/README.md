# 206. Reverse Linked List

https://leetcode.com/problems/reverse-linked-list/


# Tests

```
Input: head = [1, 2, 3, 4]
Output: [4, 3, 2, 1]
```


# Solutions

| Solution      | Time complexity | Space complexity |
|---------------|-----------------|------------------|
| **Iterative** | `O(n)`          | `O(1)`           |
| **Recursive** | `O(n)`          | `O(1)`           |
